Getting Started

Step One: Use Google Chrome

Make sure you are using Google Chrome as your web browser. Fluently uses new videoconferencing technology only available in Chrome at this time.

Step Two: "Allow" Webcam Access

Click “Allow” to give Fluently access to your computer’s webcam. This access will immediately terminate as soon as your session is completed.

Step Three: Invite a User

Enter a valid email address to invite a user to join your new session on Fluently. This invited user will receive a unique link that will expire if unused.
